# 良禽佳木跨工具生成人工评测清单

用于评估任意工具基于这套标准包产出的页面、长图、卡片或视觉稿，是否仍然“明显像良禽佳木”。

## 1. 品牌拟合

- [ ] 看起来像良禽佳木，而不是泛高端家居品牌
- [ ] 气质温润、克制、可信，没有促销感
- [ ] 中文表达自然，不是翻译腔或模板腔
- [ ] 品类不同但仍然像同一品牌，不像重新发明了一套风格
- [ ] 承载不同但仍然像同一品牌的不同交付，不像另一套模板

## 2. 结构稳定

- [ ] 页面主线清楚，用户能理解阅读顺序
- [ ] 区块选择来自白名单模块或受控载体约束
- [ ] 区块顺序合理，没有信息跳跃
- [ ] 即使去掉图片，只看文字和结构，结果也仍然成立

## 3. 信息质量

- [ ] 产品事实没有臆造
- [ ] 材质、工艺、参数等内容可判断、可对应
- [ ] 图文叙事服务结构，而不是抢结构
- [ ] 没有用装饰、图标或花哨文案掩盖事实不足

## 4. 成品感与工艺

- [ ] 看起来像真实可交付成品，而不是规范说明图、系统卡片或设计备忘
- [ ] 没有出现 `Pack`、`Page Goal`、`Preview` 等系统说明词
- [ ] 图标风格统一、克制、精细；如果精度不足，宁可减少也没有低质量图标
- [ ] 中文 UI 表达完整自然，没有过多英文元标签或控制面术语
- [ ] logo 没有丢失、替换、重绘或被纯文本品牌名冒充

## 5. CTA 与转化

- [ ] CTA 语气是邀请式，而不是催促式
- [ ] 咨询承诺解释了沟通后会发生什么
- [ ] 转化动作没有破坏整体品牌气质

## 6. 参考页使用边界

- [ ] 参考页只影响结构节奏和信息编排
- [ ] 没有照搬参考页的品牌文案、视觉符号或互动签名

## 通过标准

满足以下条件即可视为 v1 合格：

1. 输出符合对应 schema 或结构化约束
2. 没有触犯品牌禁止项
3. 没有元信息泄漏与明显工艺失真
4. 人工主观判断“明显更像良禽佳木”

## 固定回归题

除常规人工评测外，建议固定回归：

- `evaluation/high-risk-regression-cases.zh-CN.json`

这份题库优先检查：

- logo 是否会在最终视觉成品中丢失
- 带产品词的视觉请求会不会被错误路由
- 图标精度不足时是否会正确降级
- 渠道变化会不会误触发风格切换
