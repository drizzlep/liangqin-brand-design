Case Detail Template

城市住宅案例，不是图片拼贴，而是一段可被理解的空间叙事

对中文客户来说，案例详情页的作用并不只是“展示审美”，而是帮助客户理解：这个空间为什么成立、用了哪些材质、适合怎样的生活方式，以及如果自己咨询，会进入怎样的服务路径。

项目类型 城市住宅 / Residence
空间关键词 温润、安静、秩序感
适合场景 改善型住宅 / 大平层
Spatial Mood

以暖石色、柔和木感与低对比光影建立长期耐看的空间氛围。

Project Story

好的案例页，应该先说明空间判断，再展开视觉细节

这类页面最重要的不是堆满术语，而是用清楚的段落、图文顺序和节制的说明，让客户看懂空间气质、材质取舍与使用逻辑。

01 / Context

居住目标

适合注重长期居住体验的家庭，希望空间安静、稳定、耐看，不被短期风格噪声干扰。

02 / Method

空间逻辑

公共区域强调开阔与秩序，私密区域强调柔和与包裹感，让动线与情绪都自然收束。

03 / Feeling

情绪基调

不是冷静到疏离，而是在安静中保留温度，让材质与光线替代过度装饰成为主角。

Spatial Gallery

图片顺序本身，就是案例叙事的一部分

客厅主视角

先呈现整体气质、尺度关系与主材配比，再进入更细的家具与材质说明。

材质细部

近景图负责说明触感与工艺，不只是补充美图。

过渡空间

走廊、转角与边界，是最能体现秩序感与克制处理的位置。

Material Notes

材质说明不该只是参数表，而要说明它为什么适合这个空间

Wood Veneer

木饰面的温度

用于承接大面积立面与储物界面，让空间在结构理性之上保留长期生活所需的柔和感。

Stone Finish

石面的稳定

用于关键台面与背景界面，提供更坚实的中心感，同时与木质形成轻微冷暖平衡。

Textile Layer

织物的缓冲

沙发、窗帘与软装织物不追求强对比，而是帮助光线与触感更顺滑地落在空间里。

Project Facts

案例页中的项目事实，需要像目录一样清楚可检索

参数与说明应稳定对齐，帮助客户快速建立判断，而不是藏在情绪化长文里。

项目类型 住宅空间 / Residence
适配人群 重视长期居住品质的家庭
推荐重点 整体家具配置 / 材质协调 / 咨询路径
页面目标 建立信任、解释判断、引导咨询

Consultation Entry

如果客户看完案例后愿意继续了解，下一步应该自然进入咨询

最好的转化不是催促，而是让客户觉得：这个团队有判断、有方法，也愿意先理解我的空间需求。案例页的 CTA 应当承接这种信任。
