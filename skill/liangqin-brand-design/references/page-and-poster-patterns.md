# 页面与海报落地模式

这里给的是模式，不是死模板。目标是帮助输出更像良禽佳木，而不是限制为单一版式。

## 海报类

推荐层级：

1. 主标题：一句有气质但清楚的中文标题
2. 副标题：补充材质、工艺、场景或价值点
3. 主视觉：真实材质或空间触感画面
4. 小信息区：时间、地点、系列、说明
5. CTA：低调但明确

原则：

- 留白宁多勿碎
- 标题不必巨大，但必须稳
- 如果主视觉已经很强，文字要收着来

## 品牌页 / 内容页

推荐节奏：

- Hero：一句品牌判断 + 一段补充说明 + 一张强材质图
- Section：按主题分块，而不是平均切碎
- 图文模块：大图 + 短文字，避免堆字
- CTA：放在信息充分之后，而不是一上来就催促动作

## 组件 / 模块

常见要求：

- 卡片不宜过多描边和阴影
- 按钮不宜太硬、太亮、太强促销感
- 表单重点是清晰与可信，不是“做得很像艺术品”
- 导航要稳，不要用过多实验性布局破坏秩序

## 图文比例建议

- 品牌型内容：图像优先，文字辅助
- 说明型内容：图像与文字均衡
- 转化型内容：信息清楚优先，但仍保持克制
