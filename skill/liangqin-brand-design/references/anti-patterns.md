# 良禽佳木品牌反模式

下面这些方向即便看起来“也像高级设计”，也不应被当作良禽佳木风格。

## 1. 黑金重奢风

特征：纯黑背景、大面积金色描边、强烈奢侈品陈列感。

问题：良禽佳木不是炫耀式奢华，而是温润克制的长期审美。

## 2. 冷科技 SaaS 风

特征：蓝紫渐变、玻璃拟态、仪表盘感、强 UI 科技语言。

问题：会把品牌带到互联网产品审美，而不是家居与材质语境。

## 3. 高饱和营销海报风

特征：高对比跳色、促销感标题、极强 CTA 压迫感。

问题：容易损伤品牌可信度与耐看度。

## 4. 过度极简导致中文难读

特征：标题过细、间距过空、标签过短、正文像注释。

问题：看起来有设计感，但中文用户读起来费力。

## 5. 强特效与强装饰

特征：粒子、液态、炫光、复杂动效、大面积镜面和光斑。

问题：会抢走材质和内容本身的表达空间。

## 6. 廉价效果图感

特征：过度磨皮、假景深、廉价 3D 渲染、材质失真。

问题：品牌应强调真实触感，而不是假精致。
