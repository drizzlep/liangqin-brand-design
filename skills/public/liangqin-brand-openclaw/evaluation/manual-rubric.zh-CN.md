# 良禽佳木页面蓝图人工评测清单

用于评估 OpenClaw 基于本 Skill 产出的首页或产品详情蓝图是否“更像良禽佳木”。

## 1. 品牌拟合

- [ ] 看起来像良禽佳木，而不是泛高端家居品牌
- [ ] 气质温润、克制、可信，没有促销感
- [ ] 中文表达自然，不是翻译腔或模板腔

## 2. 结构稳定

- [ ] 页面主线清楚，用户能理解阅读顺序
- [ ] 区块选择来自白名单模块库
- [ ] 区块顺序合理，没有信息跳跃

## 3. 信息质量

- [ ] 产品事实没有臆造
- [ ] 材质、工艺、参数等内容可判断、可对应
- [ ] 图文叙事服务结构，而不是抢结构

## 4. CTA 与转化

- [ ] CTA 语气是邀请式，而不是催促式
- [ ] 咨询承诺解释了沟通后会发生什么
- [ ] 转化动作没有破坏整体品牌气质

## 5. 参考页使用边界

- [ ] 参考页只影响结构节奏和信息编排
- [ ] 没有照搬参考页的品牌文案、视觉符号或互动签名

## 通过标准

满足以下条件即可视为 v1 合格：

1. 蓝图符合对应 schema
2. 仅使用白名单模块
3. 没有触犯品牌禁止项
4. 人工主观判断“明显更像良禽佳木”
